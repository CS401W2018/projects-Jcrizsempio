1. Internet

The Internet is the global system of interconnected computer networks that uses the Internet protocol suite (TCP/IP)[b] to communicate between networks and devices.

2. Web

The Web is the common name for the World Wide Web, a subset of the Internet consisting of the pages that can be accessed by a Web browser.

3. How are the Internet and Web different?

The internet is a public, globally connected network of computer networks. The web is a graphical user interface to information stored on computers running web servers connected to the internet.

4. ASCII and Unicode

ASCII - Short for American Standard Code for Information Interchange, ASCII is a 7-bit character set that represents 128 characters, including numbers, letters, and some special characters

Unicode - Universal character encoding, is a standard that can represent over one million characters from many languages and scripts

5. What is UTF-8

UTF-8 is an encoding system for Unicode. It will translate any character to numbers like binary and traslate it back.

6. URL

URL - Uniform Resource Locator is the address of a unique resource on the internet.

7. Domain Name

A domain name is a string of text that maps to a IP address, used to access a website from client software.

8. HTTP

Hypertext Transfer Protocol (HTTP) is the foundation of the World Wide Web, and is used to load webpages using hypertext links.

9. HTTPS

HTTPS - Hypertext Transfer Protocol Secure, is a secure version of HTTP that encrypts communication between a client and a web server.

10. IP Address

Internet Protocol (IP) address is the unique identifying number assigned to every device connected to the internet.

11. DNS

Domain Name System or DNS turns domain names into IP addresses, which browsers use to load internet pages. Every device connected to the internet has its own IP address, which is used by other devices to locate the device.

12. W3C Recomenadations

Is a standard or guideline that the World Wide Web Consortium (W3C) endorses to promote consistency and interoperability on the web.

13. XML, XHTML, HTML, HTML5

XML - Extensible Markup Language is a markup language designed for storing and transporting data in a structured format.

XHTML- eXtensible Hypertext Markup Language is a web coding standard that combines elements of HTML (Hypertext Markup Language) and XML (eXtensible Markup Language).

HTML - Hypertext Markup Language is the standard language used to create and design web pages. It provides the basic structure and content for web documents, using a system of tags and attributes to define elements such as headings, paragraphs, links, images, and other types of content.

HTML5 - Hypertext Markup Language 5 is a markup language used for structuring and presenting hypertext documents on the World Wide Web. It was the fifth and final major HTML version that is now a retired World Wide Web Consortium (W3C) recommendation.

1. Why do we need a viewport meta tag?

The viewport meta tag is crucial for responsive web design, as it controls how a webpage is displayed on different devices. By setting the width to the device's width, it ensures the content scales properly and fits the screen. This enhances user experience by making text readable and interactive elements easily accessible. Overall, it helps create fluid layouts that adapt seamlessly to various screen sizes.

2.When you select the same tag twice, and in one of them you say "flex-direction: column;" and the other says "flex-direction: row;", which one takes priority?

If you have the same CSS rule twice with different values for the same property, the one that appears last will take priority. So, if you first write `flex-direction: column;` and then `flex-direction: row;`, the second one (`row`) will be used.

How do you change the CSS for a web page displayed on a phone?

To change the CSS for a webpage on a phone, use media queries. These let you apply specific styles based on the screen size. For example, you can set different colors or layout styles for screens that are 600 pixels wide or smaller.

  1. The form tag has an "action" attribute. What does it do?

    The `action` attribute in a 'form' tag specifies the URL where the form data should be sent when the form is submitted. It tells the browser which server or script will process the data. If left empty, the form submits to the same URL as the page containing the form.

  2. The form tag has a "method" attribute. What does it do?

    The `method` attribute in a 'form' tag defines how the form data will be sent to the server. The two most common methods are `GET`, which appends the data to the URL, and `POST`, which sends the data in the request body. The choice of method depends on the type of data being submitted and its sensitivity.

  3. What is the difference between a "POST" and a "GET"?

    GET for retrieving data and POST for submitting sensitive or large amounts of data.

Questions:

Why don't we use the border property?

We might avoid using the border property for a cleaner design, opting for padding and background colors instead. Sometimes, alternatives like box-shadow can create a more modern look. It's all about achieving the desired style and user experience!

What is the easiest way to color every other row a different color?

To color every other row, use the nth-child pseudo-class in CSS. For example, tr:nth-child(even) can set a background color for even rows, while tr:nth-child(odd) can set a different color for odd rows. This way, the rows alternate colors easily!

When should we use a table?

Only use table when you are going to present data like from excel.

CSS Basics

Explain the difference between inline, embedded, and external CSS.

CSS can be applied in three main ways: inline, embedded, and external. Inline CSS uses the style attribute directly within HTML elements for quick, specific styles. Embedded CSS is defined within a style tag in the head of an HTML document, allowing for grouped styles for that page.

Explain why you should only use external CSS.

Using external CSS is a good choice because it makes updates easier; you can change styles in one file, and they apply to all linked pages. It keeps your HTML and CSS separate, making both easier to read and manage.

What is a selector used for?

A selector in CSS is used to choose specific HTML elements to style. It tells the browser which elements to apply styles to, like changing the color or font of all paragraphs or specific buttons. By using selectors, you can effectively design and customize the look of your web page.
